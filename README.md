# Netflix EDA Project

Interactive Netflix Data Analysis project using Python, Pandas, Plotly, and Jupyter Notebook.

## Project Overview

This project analyzes Netflix Movies and TV Shows dataset using Exploratory Data Analysis (EDA) techniques.

### Features

* Data Cleaning
* Missing Value Handling
* Interactive Visualizations
* Trend Analysis
* Content Distribution Analysis

## Technologies Used

* Python
* Pandas
* NumPy
* Plotly
* Matplotlib
* Jupyter Notebook
